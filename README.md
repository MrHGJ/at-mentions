# at mentions

> - @编辑器：基于 contentEditable 实现，除了普通文本输入，还支持@选人功能。
> - @结果展示：支持人员高亮，点击人员展示人员信息卡片(可自定义内容)。
> - demo 页：左侧@编辑器输入框，右侧结果展示。

## 功能特性

**@编辑器组件**

- 支持任意普通文本输入
- 支持@选人功能
- 支持@人员高亮展示
- 支持@人员整体删除
- 支持上下键选人
- 支持关键字模糊搜索人员
- 支持 placeholder
- 支持输入文本最大字符数限制
- 支持复制粘贴

**@结果展示组件**

- 支持@人员高亮展示
- 支持@人员点击事件
- 支持@人员点击展示个人卡片

**Tip： 可以根据自己业务需求调整@人员携带的数据信息，可以自定义个人卡片展示内容**

## 运行

1. git clone 项目
2. npm i
3. npm run start
